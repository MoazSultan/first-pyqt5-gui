# first-pyqt5-gui
My first PyQt5 GUI application demonstrating button creation, event handling, and message box interaction in Python.
# PyQt5 Basics

## Overview

This repository contains beginner-friendly PyQt5 programs developed to learn the fundamentals of GUI programming in Python.

## Features

* Button creation using `QPushButton`
* Displaying messages with `QMessageBox`
* Accepting user input using `QLineEdit`
* Showing text using `QLabel`
* Handling button click events
* Updating labels dynamically

## Projects Included

1. **Button Message Box Example**

   * Displays a message box when a button is clicked.

2. **Input and Output Example**

   * Takes user input and displays it using a label.

3. **Label Update Example**

   * Changes label text when the button is pressed.

## Technologies Used

* Python
* PyQt5

## Learning Outcomes

* Understanding GUI development
* Working with widgets
* Event-driven programming
* Signal and slot mechanism in PyQt5

## How to Run

Install PyQt5:

```bash
pip install PyQt5
```

Run any file:
